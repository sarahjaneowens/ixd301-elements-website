Iron is a chemical element with the symbol (Fe) and has an atomic number of 26. It is a metal element. It is the most common element on Earth. It is the fourth most common element in the Earth's crust.

When iron is left it then turns to rusty. Rust is a reddish-brown color. Iron is a magnetic metal but not all types of iron are magnetic.
